virtual float SoS::Core::IPitchSample::getAccuracy ( ) const
pure virtual

Gets the accuracy of the sample, relative to song's notes. Valid accuracy values range from -infinity to 1.
A value of 1 means 100% accuracy. A value of 0 means the pitch missed by a whole note, a value of -1 means the pitch missed by 2 notes, etc.
Any value above 1 is invalid, which usually means the accuracy was not yet calculated.
