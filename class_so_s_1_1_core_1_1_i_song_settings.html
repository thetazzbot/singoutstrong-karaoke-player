virtual long SoS::Core::ISongSettings::getOutputLatency ( ) const
pure virtual

Gets the delay between the Song current playback time, and the actual playback.

It takes a while for the user to react to what he hears, and for the audio input to be processed.
Therefore it is useful to have a small delay between the stated playback time of the song, and when it's actually played back, for everything to sync up properly.

long The delay time in miliseconds
