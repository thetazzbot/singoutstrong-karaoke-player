bool SoS::Core::SongControl::isRecordSampleVisible ( int  sampleIndex,
int  marigin 
) const
virtual

Checks if the PitchSample at the specified index is visible at the viewTime specified in SongSettings.

Specifically, checks if the pitch is between the selected track's top and bottom note, and if the timestamp is between the viewTime and timePageLength.
Also makes sure this isn't the last sample of a section.

Parameters:
sampleIndexThe index of the sample in the PitchRecord
mariginNumber of notes that a sample's pitch can go beyond the track's top and bottom notes, that will still meet the visibility criteria.
Returns:
bool True if the sample is visible, otherwise false

void SoS::Core::SongControl::update ( )
virtual

Updates all the information about the Song, and the PitchRecord.

If the Song is playing, the function updates the current playback time in the SongSettings, and the last time a note was played in each track (which is later used to calculate the track's volume).
It also copies the PitchRecord from the AudioAnalyzer, and adjust's it to the octave of the currently selected track.

If the Song is not playing, the function does nothing.
