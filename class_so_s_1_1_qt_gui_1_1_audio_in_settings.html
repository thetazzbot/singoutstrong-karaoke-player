void SoS::QtGui::AudioInSettings::setTutorialHighlight ( int  stage)

Highlights the subwidgets for the purposes of the tutorial in the TutorialWindow.

The method sets the tutorialHighlight property to the value of the active parameter, for the deviceBox, deviceLabel, and applyButton widgets.
This allows these widgets to be highlighted when the user enters the seocnd page of the TutorialWizard.

You can set the style of these widgets while in tutorial mode by referring to the tutorialHighlight property in the skins' stylesheets.
For examlple:

#widgetID[tutorialHighlight="true"] {
...
}
Parameters:
activeThe value to which the
tutorialHighlight property of subwidgets will be set
