Detailed Description

A list widget that allows for highlighting an item independently of QListWidget's currentItem, currentRow, etc.

This is used to show which item is actually used by the application.
For instance the user can click on another song, and the item will be marked the way QListWidget does this normally.
But since a new song is not loaded until an item is doubleclicked, the currently loaded is highlighted, using additional properties stored by the HighlightList
