void SoS::QtGui::MainWindow::changeEvent ( QEvent *  e)

Called when the user activates (sets the focus on) the main window, and when it's state is changed (minimize/maximize/normal)

When the window is activate, all the sub windows are also brought to the top of the desktop.
When it is minimized all the sub windows are hidden, and when it is brought back from the minimized state the sub windows that were previously shown, are shown again.

void SoS::QtGui::MainWindow::loadSettings ( )
protected

Loads the application settings from the user's appData folder.

The location of the settings file is accessed through the QDesktopServices::storageLocation() method, and it depends on the operating system the application is ran on.
If there is no settings file then we assume this is the first run of the application and we show the tutorial.

The settings that are loaded are the windows' location, size, and visibility, and the SongSettings variables that can be set by the sub windows.

void SoS::QtGui::MainWindow::show ( )

Shows this window, and every sub window that was also visible during the prior run of the application, or before this window was minimized.

Altough evert sub window's show method is called, the SoSSubWindow only allows the windows to be shown when the allowShow flag is set.
Therefore the visibility is managed through the respective windows' checkboxes
