void SoS::QtGui::OutputSettings::setTutorialHighlight ( bool  active)

Highlights the subwidgets for the purposes of the tutorial in the TutorialWindow.

The method sets the tutorialHighlight property to the value of the active parameter, for the child widgets of the outputTracks widget.
This allows them to be highlighted when the user enters the fourth page of the TutorialWizard.

You can set the style of these widgets while in tutorial mode by referring to the tutorialHighlight property in the skins' stylesheets.
For examlple:

#widgetID[tutorialHighlight="true"] {
...
}
Parameters:
activeThe value to which the
tutorialHighlight property of subwidgets will be set

QList<int> SoS::QtGui::OutputSettings::trackIndeces
protected

List of indices (referring to the track's index in the Song class)

Since only the tracks that contain notes are displayed, the index of a track does not necessarily correspond to it's place in the outputTracks widget (or any of the subwidget lists).
So this list stores indices of all displayed tracks. For example trackIndices[0] will return the index of the first displayed track.
