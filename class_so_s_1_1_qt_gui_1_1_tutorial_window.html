Detailed Description

Window for displaying the application tutorial.

The QWizard class was causing some problems on Windows 7, so this tutorial was moved to a custom window.
This class tries to load the contents of each page from htm files in the /docs/tutorial directory. The files should be named after the page number, starting from 0 (e.g.: 0.htm, 1.htm... etc.)
