Selecting the input device

First, you need to select the input device.
The Settings window has just been opened, and the device selection box has been highlighted.
Please select the device to which your microphone is connected, and press the 'Apply' button.

If the right device was selected, you should be able to see the indicator in the Song Window react to the sounds picked up by the microphone.
If it does not, make sure the microphone is connected and properly configured in your system settings.
